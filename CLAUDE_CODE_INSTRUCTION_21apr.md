### BUG-003: תלת מימד — סיבוב מלא

## חלק 4 — צ'אט מאוחד (במקום ממשק קולי נפרד)

### CHAT-001: צ'אט בצד ימין — עם מיקרופון

**ביטול התכנון הקודם** של VoiceButton + VoicePanel overlay.
**במקום:** חלון צ'אט שנפתח בצד ימין של כל מסך.

**פעולות:**
1. צור `src/components/common/SadanChat.jsx`:
   - פאנל צד ימין (רוחב 350px) שנפתח/נסגר
   - כפתור "💬 סדן" צף בפינה — לוחצים → הפאנל נפתח
   - בתוך הפאנל:
     ```
     ┌──────────────────────┐
     │  🤖 סדן              │
     │  ──────────────────  │
     │  [היסטוריית שיחה]    │
     │  מפקד: אני רוצה...   │
     │  סדן: הבנתי, ...     │
     │  ──────────────────  │
     │  [שדה הקלדה]   🎤 📎│
     └──────────────────────┘
     ```
   - **🎤** = כפתור מיקרופון. **פעיל — Whisper + ElevenLabs עובדים!**
     - לחיצה → הקלטה מדפדפן (Web Audio API / MediaRecorder)
     - שליחה ל-backend → Whisper מקומי (גרסה גדולה) מתמלל
     - תמלול מופיע בצ'אט כהודעת משתמש
     - תשובה → ElevenLabs TTS → אודיו מושמע מהדפדפן
     - תשובה גם מופיעה כטקסט בצ'אט
   - **⌨️ שדה הקלדה** = עובד במקביל. המפקד יכול להקליד במקום לדבר — אותה תוצאה
   - **תשובות:** hardcoded עם fallback ל-Claude API:
     ```javascript
     const responses = {
       "תרגיל מחלקה": "הבנתי. רטוב או יבש?",
       "רטוב": "רמת כשירות?",
       "ג": "גודל כוח?",
       "30": "מצוין. מצאתי 3 שטחים מתאימים בגזרה שלך.",
       "בנה תרגיל": "בוא נתחיל. מה מטרת האימון?",
       // ... עוד תגובות
       "default": null  // → שלח ל-Claude API
     }
     ```
     אם אין תשובה hardcoded → שלח ל-Claude API (הסוכנים ב-backend כבר קיימים).
   - **📎** = צירוף קובץ (placeholder, disabled)

2. **הצ'אט זמין מכל מסך** — אבל לא פתוח כברירת מחדל. רק כפתור "💬".
3. **כש"בנה תרגיל מאפס" נלחץ → הצ'אט נפתח אוטומטית.**

4. **Backend endpoints חדשים:**
   - `POST /api/voice/stt` — מקבל אודיו (base64 wav/webm), מחזיר `{ text: "..." }`
     - משתמש ב-Whisper מקומי
   - `POST /api/voice/tts` — מקבל `{ text: "..." }`, מחזיר אודיו (base64 mp3)
     - משתמש ב-ElevenLabs API
   - `POST /api/voice/chat` — מקבל `{ text: "...", context: "..." }`, מחזיר `{ reply: "...", audio: "base64..." }`
     - hardcoded responses + fallback ל-Claude API
     - מחזיר גם טקסט וגם אודיו TTS של התשובה

### RULES-001: התרעות סדן

**הוסף התרעות hardcoded שמופיעות במקומות רלוונטיים:**

   ```
   ⚠️ פקח רטג לא זמין אחרי 16:00. התיאום יישלח מחר בבוקר.
   ```

   ```
   ⚠️ קצין מאשר חייב להיות מדרגת סמ"ה ומעלה.
   ```
