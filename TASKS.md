# שלב B — תשתית דמו

## TASK-003: הוספת Dependencies ומבנה בסיסי ⬜

**תלוי ב:** TASK-002 מאושר (מור בחר אופציה)

**פעולות:**
1. התקן dependencies חדשות ב-frontend:
   ```powershell
   cd frontend
   npm install react-router-dom maplibre-gl lucide-react
   ```
2. צור מבנה תיקיות חדש ב-`src/` (לפי האופציה שנבחרה):
   ```
   src/
     components/
       common/       ← Header, Button, Badge, ProgressBar
       map/          ← MapView, LayerToggle
       exercise/     ← ExerciseFileView
     screens/        ← Login, Area, Questionnaire, Plans, Exercise, Quiz, Approvals
     data/           ← mock_data.js (נתונים hardcoded)
   ```
3. הגדר Router עם 7 נתיבים:
   - `/` → Login
   - `/area` → Area
   - `/questionnaire` → Questionnaire
   - `/plans` → Plans
   - `/exercise` → Exercise
   - `/quiz` → Quiz
   - `/approvals` → Approvals
4. כל screen יחזיר placeholder — `<div>שם המסך</div>`. ללא תוכן עדיין.
5. ודא שהאפליקציה עולה (`npm run dev`) וניתן לנווט.
6. **דווח למור.**
