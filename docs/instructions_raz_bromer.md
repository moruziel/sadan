## כניסה למערכת סדן
- **בקול:** "התחל שיחה" → אשרו מיקרופון → סדן מבקש מספר אישי → אמרו: **5236521**
- **ידנית:** "כניסה ידנית" → הקלידו **5236521**

## דיבור עם סדן
העיגול למעלה מימין מציג את המצב: 🎙️ **מקשיב** (ירוק) = אפשר לדבר. **סדן מדבר** (זהב) = חכו שיסיים.
בכל מסך אפשר: "מה אני רואה?", "תסביר את המסך", "עבור לשאלון", "הפעל תלת מימד", "תראה לי את המנחת".

## וואטסאפ ושיחת טלפון אליכם (מסך אישורים)
1. אמרו לסדן: **"תתקשר אלי במספר 05XXXXXXXX"** (או ערכו את מספר היעד במסך)
2. לחצו "התקשר" ליד גורם (רטג / חי"א / GO)
3. תקבלו שיחה אמיתית — דברו עם סדן, הוא מגיב. אמרו "מאשר" + קוד כלשהו

## אם משהו נתקע
- רעננו את הדף
- קול לא עובד → בדקו שהדפדפן קיבל הרשאת מיקרופון
- עדיין תקוע → הודעה למור
