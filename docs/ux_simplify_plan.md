# תוכנית: פישוט חוויה + תיקון חפיפות (UX v2)

> **קהל:** קצינים בכירים, אנשי שטח. עיקרון: פשוט, מקצועי, אפס בלבול.
> **מקור:** פידבק מור 6.7.2026 — חפיפות אלמנטים, אין דרך ברורה לצאת/לכבות מיקרופון.

## עקרון מנחה — אלמנט צף אחד בלבד
היום צפים על המסך: עיגול קולי (ימין-עליון), כפתור 💬 (שמאל-תחתון), פאנל דמו נסתר.
שני צפים + חפיפות עם ה-header ועם כפתורי פעולה תחתונים = בלבול.
**החלטה:** העיגול הקולי הוא נקודת הכניסה היחידה לסדן. כפתור ה-💬 הצף מוסתר במובייל.

## X1 — תיקון חפיפת העיגול הקולי עם ה-header
העיגול (fixed top-right) יושב היום על אווטאר המשתמש וכפתור היציאה.
תיקון: ה-Header מפנה מקום — padding ימני, והעיגול מיושר לתוך שורת ה-header.

## X2 — הקשה על העיגול פותחת את פאנל סדן (לא מנתקת!)
היום: הקשה על העיגול מנתקת את הקול בלי אזהרה — מלכודת.
חדש: הקשה = פתיחת הפאנל (שיחה + שליטה). בתוך הפאנל:
- כפתור בולט "🎙️ נתק מיקרופון" (אדום כשמחובר) / "התחבר" (כשמנותק)
- זה עונה על "איך אני מכבה את המיקרופון"

## X3 — יציאה מהדמו: ברורה ואחידה
- כפתור "יציאה" הקיים ב-header: בנוסף לניווט למסך כניסה — מנתק את הקול ומנקה session
- פקודת קול: "צא מהמערכת" / "סיים דמו" → סדן נפרד + ניתוק + חזרה למסך כניסה
- זו התשובה ל"איך סוגרים את הדמו": כפתור יציאה או פקודה קולית. סגירת טאב תמיד עובדת.

## X4 — הסתרת כפתור ה-💬 הצף במובייל
הפאנל נפתח מהעיגול. בדסקטופ הכפתור נשאר (מסך גדול, אין חפיפה).

## X5 — סריקת חפיפות מסך-מסך (מובייל 375px + דסקטופ 1366px)
כל מסך נבדק בצילום: כניסה, בחירת שטח, מפה (2 מצבים), יומן, שאלון, מתווים,
תיק, בוחן, אישורים, סימולציה. כל חפיפה שנמצאת — מתוקנת נקודתית.

## בדיקות

- B2: צילומי מסך לפני/אחרי לכל מסך שנגעתי בו
- B3: זרימת קול: התחברות מהעיגול → פאנל → ניתוק מהפאנל → התחברות חוזרת
- B4: "צא מהמערכת" בקול → חזרה לכניסה
