## 1. מערכת "קרקע בטוחה"

### פרטים בסיסיים:
- **שם:** קרקע בטוחה
- **כתובת:** https://imunim.idf (מערכת IDF פנימית, דורשת הזדהות)
- **מפעיל:** מחלקת שטחי האש / מתא"מ צה"ל
- **גרסה/שנה:** פעילה ב-210 — ציון בפק"ל 2023, הדרכה מעדכנת בנובמבר 2025

### מטרה:
הפלטפורמה המרכזית לתיאום ואישור אימונים בשטחי אש ובשטחים פתוחים בצה"ל. **כל אימון בשטח אש חייב לקבל אישור ב"קרקע בטוחה" לפני הכניסה.**

### משתמשים:
- **מתאמי אימון** (קצינים ממדרגה ס"מ/ה ומעלה) — פותחים הקצאה
- **חמ"ל שטחי האש** — מאשר/דוחה הקצאות
- **גורמי תיאום** (11+): אוגדה 474, פולמ"ג, רפואה, עתיקות, שמירת טבע, קציסות וכו'

### קלט (מה מכניסים):
- פרטי יחידה מבקשת
- שטח אש מבוקש (קוד + שם)
- תאריך ושעות האימון
- סוג האימון (ירי/ללא ירי/חבלה/רק"מ וכו')
- מסגרת כשירות (לאיזו רמת כשירות מיועד)
- מפקד הפעילות — שם + חתימה דיגיטלית

### פלט (מה מקבלים):
- **מספר תיאום** (דוגמה: 33196, 58710/1)
- **סטטוס: "מאושרת"** — בצבע ירוק
- **הנחיות ספציפיות** מכל גורם תיאום (דוגמה מרשות הטבע והגנים):
  - "אין ירי 0.5 ומעלה, אין הצאת שרים בשטח"
  - "יש להוציא נתבים ומציאות התרשות"
  - "אין שימוש בפירוטכניקה, אין שימוש בחבלה, אין שימוש בצא"מ"
  - איש קשר: אוריה וואזה — 053-2399780

### תהליך (חמישה שלבים):
1. מתאם פותח הקצאה במחשב
2. הקצאה נשלחת לכל גורמי התיאום
3. כל גורם מאשר / מוסיף הנחיות
4. חמ"ל שטחי האש מאשר סופית
5. סטטוס משתנה ל"מאושרת"

### צילומי מסך (מהמצגת):
- **רשימת הקצאות** — עם סטטוס כל הקצאה
- **פרטי הקצאה** — מפה + גורמי תיאום + סטטוס כל גורם
- **מסמך אישור** — עם מספר תיאום ותאריך

### שכבות המידע הגיאוגרפי (GIS):
- גבולות שטחי אש (polygons)
- שמורות טבע
- חלוקות שטחות אימון
- שטחי לש"ב (מתקנים)

## 2. מערכת "תיאום בקליק"

### פרטים בסיסיים:
- **שם:** תיאום בקליק
- **סוג:** אפליקציה לנייד (Android/iOS) + ממשק ווב
- **מפעיל:** מחלקת שטחי האש / מתא"מ צה"ל
- **סטטוס:** **פיילוט — אוגדה 210 בלבד** (נובמבר 2025)
- **כרכיב:** משולב עם "קרקע בטוחה" — ההקצאה נפתחת ב"קרקע בטוחה" ועוברת לאפליקציה

### מטרה:
לאפשר תיאום אימונים **בשטחים פתוחים ללא ירי** (לא שטחי אש ממוגדרים) בצורה מהירה ונייד. מיועד לאימונים שאינם דורשים את תהליך "קרקע בטוחה" המלא.

### הבדל מ"קרקע בטוחה":
- "קרקע בטוחה" = **שטחי אש + ירי** (חובה לכולם)
- "תיאום בקליק" = **שטחים פתוחים + ללא ירי** (פיילוט, קל יותר)

### תהליך:
1. מתאם פותח הקצאה ב"קרקע בטוחה" + מסמן "תיאום בקליק"
2. הקצאה עוברת לאפליקציה עם כל הפרטים
3. גורמי התיאום מקבלים **SMS** עם בקשת אישור
4. כל גורם מאשר דרך האפליקציה (כולל הנחיות)
5. **מפקד הפעילות חותם חתימה דיגיטלית** + הצהרה:
   > "אני, מפקד/ת הפעילות, מצהיר/ה כי הפרסים שהוזנו לצורך האימון הינם מלאים ולא הסתרתי דבר, ואני מתחייב/ת לעמוד בהתחייבות הנחיות לקיום האימון."
6. הקצאה מאושרת — ממש נעלמת מהאפליקציה וחוזרת לסטטוס "מאושרת" בקרקע בטוחה

### מסכי האפליקציה (לפי המצגת):
1. **כל ההקצאות** — רשימה כוללת
2. **ממתין לאישור מגופי התיאום** — עם פרטי הקצאה
3. **ממתין לחתימת מפקד הפעילות** — שלב אחרון לפני אישור סופי
4. **חתום ע"י מפקד הפעילות** — הושלם

### מה SADAN יכול לעשות מעבר:
| מה חסר ב"תיאום בקליק" | מה SADAN יוסיף |
|-----------------------|----------------|
| בניית תיק תרגיל לפני הקצאה | תיק מוכן לפני הפתיחה |
| ניהול תהליך רב-שבועי | ניהול ציר זמן מלא |
| ממשק בעברית RTL מוכן | קיים כבר ב-SADAN |
| היסטוריה וניתוח | Dashboard היסטורי |
