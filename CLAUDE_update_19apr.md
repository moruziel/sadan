- **סדן מתקשרת בטלפון** לגורמי אישור (Twilio + ElevenLabs)
- **שתי אופציות הפעלה:** מכנית (ממשק ויזואלי) + וורבלית (שיחה)
- כפתור "🎤 דבר עם סדן" צף בכל מסך
- **בדמו — 3 שיחות טלפון חיות:**
  1. רטג (פקח) — תיאום תרגיל
  2. חי"א (מסוק) — תיאום פינוי אווירי
  3. מתא"ם (GO/NO-GO) — אישור כניסה לשטח

#### 2. סטאק קולי חדש
```
מפקד מדבר → Whisper (STT) → Claude API → ElevenLabs (TTS) → מפקד שומע

סדן מתקשרת → Twilio Voice → ElevenLabs (TTS) → פקח שומע
                                              ← Whisper (STT) ← פקח עונה
                                              → Claude API → תגובה
```

#### 3. הקצאת שטח — 3 אופציות
- **מגזרה מוקצית** — המפקד רואה גזרה שלמה, בוחר תת-שטח
- **חיפוש חופשי** — לפי מטרות/אופי שטח (הררי, בנוי...)
- **אימון דחוף** — מה פנוי עכשיו

#### 4. גאנט/יומן שטחים — מסך חדש
- Booking.com style
- לכל שטח — יומן עם תפוס/פנוי
- גרירה לשינוי תאריכים
- סדן מציעה חלופה אם תפוס

#### 5. שת"פ יחידות
- **אוטומטי:** רטוב → פינוי רכוב נוסף אוטומטית
- **ידני:** מפקד מוסיף שת"פ (מסוק, שריון, תותחנים...)
- **אינטליגנטי:** סדן מזהה כוח מקביל ומציעה שת"פ

#### 6. GO/NO-GO מתא"ם
- סצנה אחרונה בדמו
- כל האישורים ✅ → כפתור נפתח → סדן מתקשרת למתא"ם → "GO" → 🟢
